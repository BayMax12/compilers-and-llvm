**COOL**: Classroom Object Oriented Language

Designed to be implementable in a short time.

`COOL compilers >> COOL programs.`

Sample cool program:
```
class Main {
  
  i : IO <- new IO;
  main():Int { { i.out_string("Hello World!"); 1; } };
  
};


```
